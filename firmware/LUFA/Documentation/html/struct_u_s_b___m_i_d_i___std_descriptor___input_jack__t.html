Detailed Description

Type define for an Audio class-specific MIDI IN jack. This gives information to the host on a MIDI input, either a physical input jack, or a logical jack (receiving input data internally, or from the host via an endpoint).

See also
USB_MIDI_Descriptor_InputJack_t for the version of this type with non-standard LUFA specific element names.
Note
Regardless of CPU architecture, these values should be stored as little endian.
